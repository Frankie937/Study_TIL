

### 쿼리 파라미터 문법 
- 쿼리 파라미터란, URL에서 리소스 경로(path) 뒤에 추가적인 정보를 전달하고 싶을 때 사용하는, 표준화된 방법입니다.
- URL에서 ?(물음표) 뒤에 key=value 형태로 붙입니다.
- 여러 개의 쿼리 파라미터가 있을 때는 &로 구분합니다.
- ? : URL의 메인 경로(path)와, 추가적 정보를 주는 쿼리 파라미터 부분을 나누는 기준입니다.
- 쿼리 파라미터 방식은 RESTful API, 웹 브라우저, HTML 등 웹 생태계 전반에서 광범위하게 사용되는 표준 문법입니다.
- FastAPI 같은 프레임워크에서는 ?파라미터명=값 방식으로 오는 값을 자동으로 인식해서 파라미터에 넣어줍니다.
